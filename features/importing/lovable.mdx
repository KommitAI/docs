---
title: "Import from Lovable"
description: "Migrate your Lovable project to Kommit via GitHub."
---

Lovable projects automatically sync to GitHub. You can import them into Kommit by connecting the GitHub repository that Lovable created.

## Steps

1. Go to **Import** in the sidebar and select **Lovable**
2. Paste your Lovable project's GitHub repo URL, or select it from your connected GitHub account
3. Name your project and set the visibility
4. Click **Start Import**

## What gets analyzed

- **Code** — Full React/Vite source tree
- **Design and styles** — Tailwind classes, component styling patterns
- **Database schema** — Supabase schema if present

## Tips

- Your Lovable project's GitHub repo is usually named after your Lovable project
- Make sure the Lovable GitHub sync has completed before importing
