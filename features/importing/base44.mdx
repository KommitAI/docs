---
title: "Import from Base44"
description: "Migrate your Base44 project to Kommit via GitHub."
---

Base44 projects can be exported to GitHub using GitHub sync (available on the Builder plan). Once synced, you can import the repository into Kommit.

## Steps

1. In Base44, enable GitHub sync for your project (requires Builder plan)
2. Go to **Import** in Kommit and select **Base44**
3. Paste the GitHub repo URL or select it from your connected account
4. Name your project and click **Start Import**

## What gets analyzed

- **Code** — Full source tree
- **Design and styles** — Component styling and UI patterns
