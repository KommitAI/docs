Kommit integrates with the tools your team already uses. Configure integrations from **Settings > Integrations** in your dashboard.

## GitHub codebase analysis

When you connect a GitHub repository, Kommit can analyze it to generate:

- **Feature graphs** — a map of the features in your codebase
- **Dependency maps** — how modules and packages relate
- **Diffs** — what changed between analysis runs

This analysis feeds into PRD generation, giving the AI context about your existing codebase when writing requirements.
