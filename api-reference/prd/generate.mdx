---
title: "Generate PRD"
description: "Generate a product requirements document from canvas specs. Returns a streaming response."
api: "POST /v1/projects/{projectId}/prd/generate"
---

<ParamField path="projectId" type="string" required>Project UUID.</ParamField>

Reads all canvas nodes, edges, and project memory to generate a structured PRD. The response streams as **Server-Sent Events (SSE)**.

<CodeGroup>
```bash Request
curl -X POST https://api.getkommit.ai/v1/projects/a1b2c3d4-e5f6-7890-abcd-ef1234567890/prd/generate \
  -H "Authorization: Bearer km_your_api_key"
```

```text SSE response
data: {"type":"progress","stage":"reading_canvas"}
data: {"type":"progress","stage":"generating"}
data: {"type":"text","content":"# Mobile App Redesign\n\n"}
data: {"type":"text","content":"## Overview\n\nA complete redesign..."}
data: {"type":"done","prdId":"prd_b2c3d4e5","version":1}
```
</CodeGroup>

<Note>
  Generation may take 30-60 seconds depending on project complexity. The SSE stream keeps the connection alive with progress events.
</Note>
