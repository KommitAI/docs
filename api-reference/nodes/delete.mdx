---
title: "Delete node"
description: "Delete a node from the project canvas."
api: "DELETE /v1/projects/{projectId}/nodes/{nodeId}"
---

<ParamField path="projectId" type="string" required>
  Project UUID.
</ParamField>

<ParamField path="nodeId" type="string" required>
  Node UUID.
</ParamField>

<Warning>
  Deleting a node also removes all edges connected to it.
</Warning>

<CodeGroup>
```bash Request
curl -X DELETE https://api.getkommit.ai/v1/projects/a1b2c3d4-e5f6-7890-abcd-ef1234567890/nodes/n1a2b3c4-d5e6-7890-abcd-ef1234567890 \
  -H "Authorization: Bearer km_your_api_key"
```

```json Response
{
  "success": true
}
```
</CodeGroup>
